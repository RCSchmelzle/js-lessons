# JavaScript Crash Course

Interactive JavaScript lessons for complete beginners. Everything runs in the browser — no installs needed.

## Get Started

1. Click the green **Code** button above, then **Download ZIP**
2. Unzip the folder
3. Open `index.html`

## Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+Enter` / `Cmd+Enter` | Run code |
| `Tab` | Insert spaces in editor |
| `Left/Right arrows` | Navigate slides |
