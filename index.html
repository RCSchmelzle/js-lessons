JavaScript Crash Course

Interactive lessons with built-in code editors. Click any lesson to start. No installs needed — everything runs in the browser.

By the end of this crash course, you should be able to:
  • Build a small interactive web app (like a to-do list)
  • Manipulate the DOM with JavaScript
  • Handle user input and events
  • Fetch data from an API
  • Explain core JavaScript concepts in an interview
Core Lessons
Lesson 1
Foundations
[MUST KNOW] Variables, data types, arrays, objects, classes, functions, conditionals, loops.
Lesson 2
Working with Data
[MUST KNOW] .forEach(), .map(), .filter(), .find(), destructuring, spread operator.
Lesson 3
DOM & Interaction
[MUST KNOW] querySelector, innerText, classList, createElement, events. Live DOM canvas.
Lesson 4
How Apps Work
[MUST KNOW] Event-Logic-UI pattern, state, debugging the flow. The mental model.
Lesson 5
Events & Forms
[MUST KNOW] Input handling, Enter key, delete buttons, toggle complete. Live DOM canvas.
Lesson 6
Async & APIs
[MUST KNOW] setTimeout, JSON, fetch, async/await, try/catch, loading states.
Lesson 7
Scope & Closures
[GOOD TO KNOW] Scope, block scope, closures, this, arrow functions, factory functions.
Lesson 8
Bootstrap
[OPTIONAL] Adding Bootstrap, containers, spacing, buttons, inputs, lists, grid layout.
